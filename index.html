CuplSDK eclipse plugin.

The CuplSDK eclipse plugin provides Cupl functionality from within eclipse. Apart from "standard" Cupl functionality, the eclipse plugin contains a Cupl project nature and also adds a new Cupl perspective.The plugin makes use of the all core funcionalities of the WinCupl.

CuplSDK eclipse Update Site

An eclipse update site is now available at http://psembedded.github.io/CuplSDK/update.

Install the CuplSDK eclipseplugin with the following steps:

1. Select Help > Install New Software...

2. In the Work with field, type http://psembedded.github.io/CuplSDK/update and click Add...

3. Enter the following in the dialog that appears:

4. Check the CuplSDK checkbox and click Next. Then follow the dialogs to install the CuplSDK feature

The CuplSDK Perspective

Once installed, open the CuplSDK Perspective by using the standard Window/Open Perspective/Other... Command and selecting "Cupl" from the list. The perspective will open 3 views:

* A "CuplSDK Layout" view to the left containing device layout

* A "CuplSDK Symbols" view to the bottom containing symbols of the generated file.

The CuplSDK Project Wizard

Once installed, create a Cupl project by using the standard File/New/Project... Command and selecting "Cupl" from the list. The wizard alows create 3 templates project:

* A "ATF22V10" project based

* A "ATF750C" project based

* A "ATF15xx" project based
